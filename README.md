# WakeMate

A simple application built with Vue + Tauri to prevent your computer from sleeping.

### Commands

Dev:

```
npm run tauri dev
```

Build:

```
npm run tauri build
```

Building on linux:

```
NO_STRIP=true npm run tauri build
```
